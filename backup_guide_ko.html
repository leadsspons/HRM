📂 1부 · 백업 위치 찾기

Hostinger가 계정을 자동 백업합니다 — 여는 위치는 여기
  1. hPanel 로그인 후 Websites → 사이트의 Dashboard 로 들어갑니다.
  2. 왼쪽 메뉴에서 Files → Backups 를 클릭합니다. Dashboard› Files›Backups
  3. 백업은 두 종류입니다: Files backups(파일)와 Database backups(DB).
    hpanel.hostinger.com › Backups
    🗂️ Files backups
    웹사이트 파일 (HTML, PHP, 이미지…)
    Select
    🗄️ Database backups
    MySQL 데이터베이스 (.sql)
    Select

⬇️ 2부 · 백업 다운로드

내 컴퓨터에 사본을 보관하세요
  1. Files backups 에서 Select 을 누릅니다.
  2. 원하는 날짜(보통 최신)를 고르고 → Next step 을 누릅니다.
  3. Download all files 를 누릅니다. .tar.gz 형식으로 받아집니다.
  4. DB는 Database backups 를 쓰며 .sql.gz 형식으로 받아집니다.
💡 전체 대신 특정 폴더·파일 하나만 골라 받을 수도 있습니다.

♻️ 3부 · 백업 복원

운영 중인 사이트를 이전 시점으로 되돌리기
  1. 파일: Files → Backups → Files backups → Select → 날짜 선택 → Next step.
  2. 필요한 파일·폴더를 고르고 Restore files 를 누릅니다.
  3. DB: Database backups → 날짜 선택 → Show databases.
  4. 데이터베이스를 선택하고 Restore 를 누른 뒤 완료까지 기다립니다.
⚠️ 복원은 현재 파일/DB를 덮어씁니다. 되돌릴 수 있도록 지금 상태를 먼저 다운로드해 두세요.

⏳ 참고 · 백업 보관 기간

보관 기간은 호스팅 플랜에 따라 다릅니다
  1. 주간 백업은 6주 보관됩니다.
  2. 일간 백업(상위 플랜)은 7일 보관됩니다.
  3. 이후 오래된 백업은 영구 삭제되므로 중요한 백업은 제때 받아두세요.
